# Статус

## Что делает код

Веб-приложение с графическим интерфейсом для управления статусом.

Возможности:

* Включение статуса (**ON**) с зелёной индикацией;
* Выключение статуса (**OFF**) с красной индикацией;
* Сохранение текущего статуса в файл `status.txt`;
* Автоматическая загрузка сохранённого статуса при открытии страницы.

## Как запустить

### Требования

* Python 3.x
* Flask

```bash
pip install flask
```

### Запуск

```bash
python app.py
```

После запуска приложение будет доступно по адресу:

```text
http://localhost:5000
```

## Использование

После открытия страницы доступны две кнопки:

* **«Включить»** — устанавливает статус `ON`;
* **«Выключить»** — устанавливает статус `OFF`.

Текущий статус отображается в центре страницы.

## Хранение статуса

Текущее значение сохраняется в файле `status.txt`.

При перезапуске сервера сохранённый статус автоматически загружается и отображается на странице.
