# Роборука

## Принцип работы

Роборука представляет собой систему с тросовым приводом и обратной связью по натяжению.

### Схема работы

1. Пользователь нажимает кнопку **ON/OFF** на сайте.
2. Сайт отправляет запрос на контроллер.
3. Контроллер передаёт команду сервоприводу.
4. Сервопривод начинает натягивать тросы.
5. Натяжение тросов приводит в движение шарнирный механизм, сгибая пальцы роборуки.
6. Датчик натяжения контролирует усилие.
7. Датчик передаёт обратный сигнал контроллеру.
8. Контроллер принимает решение:

   * продолжать натяжение тросов;
   * остановить сервопривод при достижении необходимого усилия.
9. После отключения привода положение руки сохраняется до получения следующей команды.

## Особенности конструкции

* Тросовый механизм передачи усилия;
* Шарнирные соединения для движения пальцев;
* Пружины для возврата элементов конструкции;
* Управление через веб-интерфейс;
* Обратная связь по натяжению для более безопасной работы механизма.
